# カルマン渦列解析コード
## 作成物の説明（目的・背景）: 
学部実験において、流体力学における重要な現象である「カルマン渦」の挙動を高精度かつ定量的に評価するためのプログラム。トレーサー（コショウ）を浮かべた水面の流体動画から、視覚的な情報を数値化・解析することを目的としている。

## 自身が担当した役割: 
動画データの前処理、画像処理による流速ベクトル場の計算、および渦中心の自動検知アルゴリズムの実装・検証を単独で担当した。

## 直面した課題と解決方法:
【課題】 トレーサー（コショウ）の数は有限であるため、速度場から算出した渦度を用いて計算する方法では、統計ノイズにより、渦の中心の座標が正確に定まらないことが課題であった。
【解決策】 渦の中心の周りの速度ベクトルが、幾何学的な「円」に近いことを利用し、独自の評価関数を設計することで、正確な渦の中心の座標を求めることができるようになった。
